# Phase 10A - Infantry Sandbox Foundations

Adds a squad-level infantry test loop:

- Spawn friendly rifle squads and enemy rifle squads from the debug panel.
- Select friendly squads with Alt+click near the squad or with Select Next.
- Right-click the map to issue a Secure Area order.
- Squads move in formation, secure the objective, and occupy nearby cover.
- If an enemy squad is detected before arrival, the squad stops, seeks cover, digs in, and exchanges fire.
- Infantry stats show selected squad state, mission, morale, suppression, and alive soldiers.

This is intentionally a sandbox AI foundation, not final combat simulation.
